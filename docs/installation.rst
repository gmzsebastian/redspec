.. _install:

Installation
============

The ``redspec`` package can be installed directly from the Github source.

From source
-----------

You can also install the ``redspec`` package directly from the source on `GitHub <https://github.com/gmzsebastian/redspec>`_.

.. code-block:: bash

    git clone https://github.com/gmzsebastian/redspec.git
    cd redspec
    python -m pip install -e .
